## Run Locally

Clone the project

```bash
  git clone https://github.com/avs-7955/yt-playlist-download.git
```

Go to the project directory

```bash
  cd yt-playlist-download
```

Run the script

```bash
  python main.py <videolink>
```

Example:

```bash
  python main.py https://www.youtube.com/watch?v=TQXMl4GycD0
```

The download starts.
